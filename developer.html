UI Developer

|
Full time
Apply
Job summary:

As a UI Developer, you will be responsible for designing and implementing user interfaces for web and mobile applications. You will collaborate closely with UX designers and front-end developers to create visually appealing and user-friendly interfaces.

Key responsibilities:
  1. User Interface Design: Collaborate with UX designers to translate wireframes and prototypes into visually engaging user interfaces.
  2. Front-end Development: Write clean, efficient, and responsive HTML, CSS, and JavaScript code to implement user interfaces.
  3. Cross-Browser Compatibility: Ensure that user interfaces work consistently across various web browsers and devices.
  4. Responsive Design: Implement responsive design principles to ensure that interfaces adapt to different screen sizes and orientations.
  5. Performance Optimization: Optimize user interfaces for speed and performance, minimizing page load times.
  6. Accessibility: Ensure that user interfaces are accessible to individuals with disabilities and comply with relevant accessibility standards (e.g., WCAG).
  7. Collaboration: Work closely with front-end developers, back-end developers, and other team members to integrate user interfaces with the overall application functionality.
  8. Testing and Debugging: Conduct thorough testing and debugging of user interfaces to identify and resolve issues.
  9. Version Control: Use version control systems (e.g., Git) to manage and track changes to code.
  10. Stay Current: Stay up-to-date with the latest web development trends, technologies, and best practices in UI/UX design.
Qualifications:
  • Bachelor's degree in Computer Science, Web Development, or a related field (or equivalent work experience).
  • Proven experience as a UI Developer or similar role.
  • Proficiency in HTML, CSS, and JavaScript.
  • Familiarity with front-end libraries and frameworks (e.g., React, Angular, Vue.js).
  • Experience with responsive and mobile-first design.
  • Knowledge of web accessibility standards and best practices.
  • Strong problem-solving and debugging skills.
  • Excellent communication and teamwork skills.
  • Attention to detail and a strong commitment to delivering high-quality work.
  • Experience with design tools such as Adobe XD, Sketch, Figma, or similar is a plus.
Qualifications:
  • Knowledge of user experience (UX) design principles.
  • Experience with animation libraries (e.g., GreenSock Animation Platform, CSS animations).
  • Understanding of back-end development and API integration.
  • Experience with prototyping and wireframing tools.
